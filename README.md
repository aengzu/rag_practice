### How to Use  
1. **Create a virtual environment**  
   ```bash
   conda env create -f environment.yml
   ```

2. **Activate the environment**  
   ```bash
   conda activate llms
   ```

3. **Create a `.env` file in the root directory**  
   - Add your `OPENAI_` credentials.

4. **Open `jupyter lab`**
   - `jupyter lab`
